# 🎨 Pixi - art

A custom Pixel Art Editor that provides the essential tools needed to create simple pixel art drawings directly in the browser.

## ✨ Features

- 🖌️ Draw by clicking on individual pixels.
- 🎨 Color palette for selecting different drawing colors.
- 🧹 Erase pixels by clicking on a painted cell.
- 📐 Grid-based canvas for precise pixel art creation.
- ⚡ Fast and responsive interface.
- 🌐 Runs entirely in the browser (no installation required).


## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
